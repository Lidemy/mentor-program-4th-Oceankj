## 請以自己的話解釋 API 是什麼

提供使用者跟網頁上的資料互動的方法，不同於npm直接提供開源的程式碼，且可供編輯。API提供的只有開發者所限定的功能，對於使用者而言，雖然少了自由編輯的，也沒有辦法了解底層的運作，但優點就是若運行一些較為龐大的程式，可以大大減少本機對於硬體設備的要求。

####500 Internal Server Error : 
通用錯誤訊息，伺服器遇到了一個未曾預料的狀況，導致了它無法完成對請求的處理。
